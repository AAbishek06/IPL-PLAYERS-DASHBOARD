# IPL-PLAYERS-DASHBOARD
IPL Player Analytics Dashboard 🏏
1. Overview

IPL Player Analytics Dashboard is a Python-based mini data analysis project that provides insights into IPL player performance using Pandas, NumPy, and Matplotlib. The dashboard offers an interactive menu to analyze player statistics and visualize data through charts.

2. Features
View Top Run Scorers
Analyze Highest Strike Rates
Team-wise Run Contribution Analysis
Overall Performance Statistics
Interactive Menu-Driven Interface
Data Visualization using Bar Charts and Pie Charts
3. Technologies Used
Python
Pandas
NumPy
Matplotlib
4.Dataset Structure

The CSV file should contain:

Player,Team,Runs,Balls
Virat Kohli,RCB,741,500
MS Dhoni,CSK,320,220
Rohit Sharma,MI,550,400
5. How to Run
pip install pandas numpy matplotlib
python ipl_dashboard.py
6. Dashboard Modules
Top Run Scorers

Displays players ranked by total runs with a bar chart visualization.

Highest Strike Rate

Calculates and compares player strike rates using:

Strike Rate = (Runs / Balls) × 100

Team-wise Runs

Shows total runs scored by each team and visualizes contribution using a pie chart.

Overall Statistics
Average Runs
Highest Runs
Lowest Runs
Average Strike Rate
